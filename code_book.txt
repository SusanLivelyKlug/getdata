# getdata Codebook for column labels

"subject" : The subject's ID code
"activity" : The activity measured (one of six)

### The rest of the columns are labeled with the codes that came
with the original data set.  Please see the features_info.txt and
the links to the online documentation in the README.md for details 
from the original data gatherers.

The columns in the final tidy_data.txt are actually the averages
of the original data for each activity, per subject ID.
